# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately by opening a
[security advisory](https://github.com/wingnut128/wa-docs/security/advisories/new).

Do **not** open a public issue for security reports.

You can expect an initial response within a few days. If the issue is
confirmed, we will coordinate a fix and a disclosure timeline with you.

## Supported Versions

This project is developed against the `main` branch. Only the latest commit
on `main` receives security fixes; there are no released versions or
backports.
